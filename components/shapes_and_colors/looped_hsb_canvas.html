This project has been made to understand the difference between RGB and HSB as well as how to utilise it appropriately. To program this, the colorMode() must be set to HSB while its hue and saturation are determined by the width and height of the canvas. A nested for() loop is then created to evenly spread out the colours. 


                        function setup() {
                            // RGB to HSB:
                            colorMode(HSB, width, height, 100); 
                        }

                        function draw() {
                            // HSB Canvas:
                            let stepX = 15;
                            let stepY = 15;

                            for (let y = 0; y < height; y = y + stepY) {
                                for (let x = 0; x < width; x = x + stepX) {
                                    noStroke();
                                    fill(x, height - y, 100);
                                    rect(x, y, stepX, stepY);
                                }
                            }
                        }
